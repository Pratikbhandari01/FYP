# Increment Release Checklist

## Pre-Release

- [ ] Increment scope is complete.
- [ ] No unresolved high-severity defects.
- [ ] Migrations reviewed and tested.
- [ ] URL and template routing checked.
- [ ] Role-based access checked (customer/agent/admin).

## Quality

- [ ] Core flows tested:
  - [ ] Register/Login
  - [ ] Agent hotel and room management
  - [ ] Room type management
  - [ ] Booking flow
- [ ] Regression checks executed.
- [ ] Currency and labels are consistent (NPR).

## Documentation

- [ ] Increment template filled and archived.
- [ ] Release note prepared.
- [ ] Any config change documented.

## Deployment

- [ ] Backup plan ready.
- [ ] Rollback plan validated.
- [ ] Post-deploy smoke test complete.

## Sign-off

- [ ] Developer sign-off
- [ ] Reviewer sign-off
- [ ] Product/Owner sign-off
